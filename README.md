# 🛠 Tecnologias utilizadas
## ⚙️ Back end
- Java
- Spring Boot
- JPA / Hibernate
- Maven
- Banco de dados: H2

# :rocket: Como executar o projeto
## 🎲 Back end
Pré-requisitos: Java 11

```sh
# clonar repositório
git clone https://github.com/MaxsuelSantos/client-api.git

# entrar na pasta do projeto back end
cd backend

# executar o projeto
./mvnw spring-boot:run
```
